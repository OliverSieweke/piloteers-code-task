Internal API

This section describes the internal API methods used to reatrieve location data.

getLocationList

Retrieves the list of all locations.

getLocationList(sort: number): Array
Parameters
sort (number = 1) = 1 Sort order. By default the locations are sorted in ascending alphabetical order. Set to -1 to sort in descending order.
Returns
Array: Array of alphabetically sorted locations

getLocationDetails

Retrieves a location's details, including the calculated distance to the office.

getLocationDetails(name: string): Array<{latitude: number, longitude: number, additionalData: any, distanceToOffice: number}>
Parameters
name (string) Location name (should not include a .json extension)
Returns
Array<{latitude: number, longitude: number, additionalData: any, distanceToOffice: number}>: The distance to the office is given in kilometers

Utilities

This section lists the various utility functions

validateLocation

Validates location JSON documents according to the schema location.schema.json.

validateLocation(filename: string): Promise<(boolean | Error)>
Parameters
filename (string) With .json extension
Returns
Promise<(boolean | Error)>: Returns a boolean indicating the validity or an error object if the location could not be found or parsed.

calculateDistanceToOffice

Calculates the flight distance to the office in kilometers using the Haversine Formula.

calculateDistanceToOffice(locationLatitude: number, locationLongitude: number): number
Parameters
locationLatitude (number)
locationLongitude (number)
Returns
number: Distance to the office in kilometers

formatPromise

Formats the promise result.

formatPromise(promise: Promise): Promise<Array>
Parameters
promise (Promise)
Returns
Promise<Array>:

  • Resolves to [null, result] on original promise resolution.

                       * Resolves to [err, null] on original promise rejection.
